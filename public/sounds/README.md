# Ambient sound loops

Most of the mixer needs **no files** — Rain, Ocean, Wind, Stream, Fireplace and
White/Pink/Brown noise are all **synthesised live** in the audio engine (original,
seamless, zero-asset).

Only these 7 are recording-dependent. Drop seamless-loop MP3s here with these
exact filenames to light them up (they stay greyed until present):

thunder.mp3  forest.mp3  birds.mp3  night.mp3  cafe.mp3  keyboard.mp3  train.mp3

Sources (royalty-free — check each licence): Pixabay Sounds, Freesound (CC0),
Mixkit. Aim for ~1–3 min seamless loops, 128–192 kbps.
